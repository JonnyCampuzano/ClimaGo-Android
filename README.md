# ClimaGo

ClimaGo es una aplicación Android para consultar información climática, donde tenemos las opciones de 
buscar ciudades y guardar ubicaciones favoritas.

## Tecnologías

- Kotlin
- Jetpack Compose
- Navigation Compose
- Room
- DataStore
- MVVM
- StateFlow

## Avance de la semana 1

- Proyecto Android creado.
- Repositorio GitHub configurado.
- Estructura inicial del proyecto.
- Pantallas base creadas.
- Navegación implementada.
- Room configurado.
- Preferencia de modo oscuro con DataStore.
